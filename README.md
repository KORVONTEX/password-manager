# 🔐 Password Manager

Простой и безопасный менеджер паролей с шифрованием **AES-256**.  
Все пароли хранятся в зашифрованном виде. Доступ к ним открывает только **мастер-пароль**.

---

## ✨ Возможности

- 🔒 Шифрование **AES-256** + **PBKDF2** (100 000 итераций)
- ➕ Добавление, поиск, удаление паролей
- 🔍 Поиск по части названия сайта
- 🎲 Генератор случайных надёжных паролей
- 📋 Копирование пароля в буфер обмена
- 🔑 Смена мастер-пароля (с перешифровкой всех данных)
- 💾 Экспорт / импорт паролей в JSON
- 🎨 Цветное консольное меню

---

## 🚀 Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone https://github.com/KORVONTEX/password-manager.git
cd password_manager
```
### 2. Установить зависимости
```bash
pip install -r requirements.txt
```
### 3. Запустить программу
```bash
python manager.py
```
ИЛИ открыть *manager.exe* из папки *(антивирус может ругаться!)*

---

## 🔐 Первый запуск

### 1. Придумайте **мастер-пароль** и запомните его

### 2. Повторите мастер-пароль для подтверждения

### 3. Программа создаст зашифрованный файл *passwords.enc*

⚠️ ВНИМАНИЕ: Мастер-пароль невозможно восстановить!
Потеряете его — потеряете все пароли навсегда.

## 🛠️ Используемые технологии

- **Python 3.13** — язык программирования
- **cryptography** — AES-256 шифрование, PBKDF2 (100 000 итераций)
- **pyperclip** — копирование паролей в буфер обмена
- **colorama** — цветной вывод в консоль
- **PyInstaller** — сборка в исполняемый .exe файл
- **Стандартные модули:** os, json, base64, random, string, getpass

### Алгоритмы шифрования
- **AES-256** — симметричное шифрование паролей
- **PBKDF2** — преобразование мастер-пароля в ключ
- **SHA-256** — хэширование

---

## ⚠️ Безопасность

- Все пароли хранятся в зашифрованном виде (AES-256)
- Мастер-пароль не хранится — он преобразуется в ключ через PBKDF2
- Даже при потере файла *passwords.enc* злоумышленник не сможет его расшифровать без мастер-пароля
